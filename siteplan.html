BeSelfless NGO - Website Plan

Site Name

BeSelfless NGO

This name was selected to reflect the mission of selflessness in supporting students through tuition assistance and tech centers across Uganda.

Site Purpose

The BeSelfless NGO website will serve as a hub to provide information about the organization's mission, facilitate donations, showcase tech center resources, and allow student registration and community involvement.

Scenarios

  • How can I register as a student to access tech center resources?
  • Where can I find information about volunteering opportunities?

Color Schema

  • Primary Color: #F9DC5C - Used for headings, buttons, and accents.
  • Secondary Color: #FAE588 - Used for section backgrounds.
  • Accent Color: #FCEFB4 - Used for links and hover effects.
  • Background Color: #FDF8E1 - Used for the overall page background.
  • Text Color: #2D3748 - Used for all text content.

Typography

  • Merriweather: Used for body text and paragraphs.
  • Libre Baskerville: Used for headings and navigation menus.
  • Noto Serif: Used for testimonials and quotes.
